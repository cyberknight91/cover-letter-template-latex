# Professional LaTeX Cover Letter Template

Minimal and professional LaTeX template designed for writing cover letters.

## Features

- Clean professional layout
- Easy to customize
- Compatible with pdfLaTeX
- Suitable for professional applications

## Usage

1. Edit `cybersecurity-cv-template.tex`
2. Replace personal information
3. Compile using LaTeX

## Requirements

- TeX Live or MiKTeX
- TeXstudio or any LaTeX editor
